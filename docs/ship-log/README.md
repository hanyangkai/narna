# NARNA Ship Log

Public build journal — one short entry per day. Goal: **prove progress**, not hype.

## Format (15–30 min/day)

1. Copy [`_template.md`](./_template.md) → `YYYY-MM-DD.md`
2. Fill **Shipped**, **Proof**, **One-liner**, **Tomorrow**
3. Commit: `ship-log: YYYY-MM-DD — <short title>`
4. Post the **One-liner** to X/LinkedIn + [GitHub Discussions](https://github.com/hanyangkai/narna/discussions) (category: Ship log)

## Rules

- Lead with **UGS + Packages + GU**, not generic "agent passport"
- Every entry needs **proof** (test, URL, curl, screenshot)
- Skip days with nothing real — don't post filler
- Link `narna.org` or a spec path when relevant
